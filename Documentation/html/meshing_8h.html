Header of functions to handle mesh files (read & write)

Warning
The mesh files have to be in a specific format
The domain is supposed to be 2D and rectangular

Read a mesh file and assign parameters values, coordinates, and edges references numbers.

Warning
Every parameter to be assigned and the two 2 dimensional tabs (coords, gNb_node, refEdg) are passed as pointers .
p_coords, p_refEdg and p_nod_gNb are supposed to be empty pointers that aren't allocated yet
Parameters
input_meshmesh file's relative path
type(pointer to) type of the element (1: quadrangle, 2:triangle)
n_nod(pointer to) total number of nodes
p_coords(pointer to) nodes' coordinates
n_elem(pointer to) number of element
p_nod_gNb(pointer to) matrix that contains nodes' global number for each element
n_nod_elem(pointer to) number of nodes per element
n_edg_elem(pointer to) number of edges per element
p_refEdg(pointer to) matrix that contains edges reference number for each element
Returns
0: successfully reading, -1 : error when opening the mesh file
